Chromesome is a Google Chrome extension, a set of command-line commands for the Developer Toolbar.

You can see it in action on the google.com

h2. Currently supported commands


h3. cd (nodeDescription)

Changes the current node to the first childnode that matches the nodeDescription.
<pre>
<code>> cd(body)</code>
</pre>

<hr/>

h3. ls

Lists the childnodes of the current node.
<pre>
<code>> ls</code>
</pre>

<hr/>

h3. pwd

Returns the current node.
<pre>
<code>> pwd</code>
</pre>

<hr/>

h3. scripts

Shows all the script elements found in the DOM.
<pre>
<code>> scripts</code>
</pre>
